
# SmartSite Website Setup Guide (Updated with Images & Branding)

## How to Open in VS Code
1. Open VS Code
2. Click File > Open Folder
3. Select the 'SmartSite_FullWebsite_Updated' folder
4. Open 'html/index.html' in a browser

## How to Modify
- Edit the HTML in 'html/index.html' and pages in 'pages/'
- Update styles in 'css/style.css'
- Add interactivity in 'js/script.js'

## How to Host Online
1. Upload the folder to Netlify, Vercel, or a web hosting service
2. Deploy using GitHub Pages for free hosting
